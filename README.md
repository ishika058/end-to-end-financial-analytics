# Financial Data Automation Dashboard

## Project Overview
This project automates the collection, processing, and analysis of financial survey files received daily through email.

The workflow uses Outlook Rules, Google Apps Script, Google Drive API, Power BI, and Power Query to automate reporting and dashboard generation.

## Workflow Architecture
Outlook → Gmail → Google Apps Script → Google Drive → API → Power BI

## Technologies Used
- Power BI
- Power Query
- Google Apps Script
- Google Drive API
- Outlook Rules
- Excel

## Features
- Automated file ingestion
- Dynamic dashboard refresh
- Data cleaning automation
- Credit utilization analysis
- Age group analysis
- LTV score calculation
- Loan distribution insights
- Customer segmentation
